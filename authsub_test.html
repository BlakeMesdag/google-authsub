Google Authsub Test

These pages test the Ruby google-authsub library and explain how to use it.

STEP 1: Request a token from Google

For a web app to access Google's services. We need to get an authorization token from Google.

We will create our GoogleAuthSub object something like this:

auth = GoogleAuthSub.new(:next_url=>"http://www.schedy.com/test2.html, :session=>false, :secure=>false, :scope=>"http://google.com/calendar/feeds")

The url to redirect to is found from auth.request_url. It is then up to the application to perform this redirect.

If the request is successful and the user logs in correctly, the user will be redirected back to the auth.next_url
